# Le Céleste - Restaurant Gastronomique

## Concept
Site web immersif et premium pour un restaurant gastronomique haut de gamme, offrant une expérience utilisateur mémorable avec des animations fluides et un design sombre/chic.

## Stack Technique
- Frontend : next
- Backend : node_express
- Base de données : postgresql
- CSS : tailwind

## Pages
- **/** — Accueil: Page d'accueil avec hero section et vidéo de fond
- **/menu** — Menu: Page interactive du menu avec catégories et animations
- **/reservations** — Réservations: Page de réservation en ligne avec formulaire
- **/gallery** — Galerie: Galerie photo avec lightbox et effets de parallaxe
- **/chef** — Notre Chef: Page de biographie du chef avec photo
- **/contact** — Contact: Page de contact avec coordonnées et carte interactive

## API
- `POST /api/auth/login` — Authentification utilisateur
- `POST /api/reservations` — Créer une réservation
- `GET /api/menu` — Récupérer les éléments du menu
- `GET /api/testimonials` — Récupérer les témoignages
- `GET /api/gallery` — Récupérer les images de la galerie
